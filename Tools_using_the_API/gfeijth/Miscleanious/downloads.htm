System Tray Notifier

The System Tray Notifier runs on your PC in the background and will notify you with popups, sounds, etc. whenever PRTG discovers changes in your network.

Simply download and install the software. Start the software and you will see a PRTG icon in the Windows System Tray in the lower right corner of your screen. To configure the software please right click the icon and choose "Options". Enter your account credentials and the DNS name of your server.

The program will now run in the background and will show a popup, play a sound or show a blinking tray icon to notify you about alarms, messages or todos.

SNMP Helper (Freeware)

Paessler SNMP Helper enables PRTG to collect in-depth performance information from Windows servers and workstations. Up to several thousand PC parameters and performance counters can be monitored with just a few mouse clicks. There are three different available variations:

  • Freeware Edition: Supports monitoring of memory, disks, network, and processors and comes free with the PRTG installer
  • Pro Edition: Adds more than 2000 performance counters for servers and workstations running Windows 2000, XP or 2003
  • Pro Extensions: Are available for in-depth monitoring of MS Exchange Server, MS ISA Server, MS SQL Server and MS Biztalk Server

The Freeware Edition supports about 80 performance counters and can be downloaded below. You must install SNMP Helper on the machine(s) you want to monitor. Run this setup on all the systems you want to monitor. Afterwards you can monitor the additional system parameters by simply adding new SNMP library sensors.
